# taooffoo.com

- Need ruby 2.5 or better (latest 2.7.2 works great)
> https://linuxize.com/post/how-to-install-ruby-on-centos-7/
> rbenv install 2.7.2
> rbenv global 2.7.2


> gem install jekyll bundler
> bundle install
> bundle exec jekyll build

> ./deploy.sh live
